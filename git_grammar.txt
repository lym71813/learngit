# git diff 可查看git版本之间的区别
# git log 查看git历史记录
# git reset --hard ***** 将文件恢复至****版本
# git reflog 查看git执行的所有history操作
# git remote add origin git@github.com:name/gitrepogit 添加Github账户及git库
